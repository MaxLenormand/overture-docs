---
title: GERS Demonstrations
---

Overture member companies have created several GERS prototypes and examples to help you get started with data matching and Overture ID assignment. These demonstrations cover Overture's buildings, divisions, places, and transportation themes. 

- Esri published a blog post about matching Overture buildings data with FEMA's USA structures: https://www.esri.com/arcgis-blog/products/arcgis-online/mapping/enriching-overture-data-with-gers/
- Esri shared a feature layer with Overture buildings and FEMA's USA structures (this is Overture ID to FEMA uuid match): https://www.arcgis.com/home/item.html?id=a1ebe4c138444ac5b8bf2610d544ee0c
- Esri shared a feature layer with Overture counties, Census county geometries, and Census socioeconomic data (this is an Overture ID to FIPS code match): https://www.arcgis.com/home/item.html?id=c640a11058e34316a69f6a1e74049f9e
- Safegraph published a blog post about Placekey facilitating matches between Overture POIs and National Provider Identifier (NPI) data: https://www.placekey.io/blog/joining-overture-and-npi-datasets
- Safegraph shared notebook tutorial for matching places data: https://colab.research.google.com/drive/17BimmoiW4bqpyBb0-MdK2z9MwcSVAb-8?usp=sharing
- TomTom published a blog post about the importance of GERS in Overture's transportation data model: https://engineering.tomtom.com/overture-transportation-network-linear-referencing/
- Overture's Transportation Task Force shared examples of matching GPS traces to Overture transportation data: https://github.com/OvertureMaps/schema/tree/main/gers/examples/python
