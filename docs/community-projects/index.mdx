We invite you to explore this list of publicly available tutorials, blog posts, and projects created by the Overture Maps developer community. These projects are generated and maintained by the community, and are not owned or maintained by the Overture Foundation. Want to submit your own project? Please review our contributor guidelines or reach out to us via email. 

The data is available at the following locations:

| Project | Creator | Link |
| ------ | -------- | -------- |
| Exploring the Overture Maps places data using DuckDB, sqlite-utils and Datasette | Simon Willison | `https://til.simonwillison.net/overture-maps/overture-maps-parquet` |
| Creating map tiles from Overture Data | Brandon Liu | `https://github.com/bdon/overture-tiles` |
| Overture Buildings and Places (Cloud-Native Geo experiments) | Chris Holmes | `https://beta.source.coop/repositories/cholmes/overture/description/` |

Read the [Overture Data Schema Reference](https://docs.overturemaps.org/reference) to understand the Overture Data Schema and see the [`overturemaps/data`](https://github.com/OvertureMaps/data) Github repository for information on the latest release.
